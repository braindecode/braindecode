Getting starting
^^^^^^^^^^^^^^^^^

Learn how to use braindecode by the examples. There are
several ways to learn and understand how a library works.
Here at braindecode, we encourage hands-on mode.

See which tutorial best fits the approach
you want and try to reproduce it on the examples data or on your data :)
